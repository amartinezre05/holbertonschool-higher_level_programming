Module 1-my_list
================

For the use of the function 'print_sorted'
-----------------------------------------

>>> MyList = __import__('1-my_list').MyList

Examples:
>>> new_list = MyList([])

>>> list_1 = [1, 4, -3, 5]

[-3, 1, 4, 5]

>>> list_1 = [1, 4, -3, 5]

True
